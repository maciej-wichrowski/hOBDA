Reads a set of training examples from a CSV string.

All lines will be grouped into matrices by their first value, REQ_ID. At the moment the first occurrence of a new REQ_ID is assumed to be the picked supplier.

Expected input format: 

 REQ_ID, SUPPLIER_ID, x1, x2, x3
 REQ_ID, SUPPLIER_ID, x1, x2, x3
